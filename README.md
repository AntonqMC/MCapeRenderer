# YOU NEED NODE.JS TO USE IT!
## Tutorial:
1. download it
2. drag the cape, background and skin to `assets` folder, and rename them:
- background - `background.png`
- cape - `cape.png`
- skin - `skin.png`
3. run `setup.bat`
4. that should create `start.bat`, run it
5. enjoy
